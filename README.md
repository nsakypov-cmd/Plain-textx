# STUDENT.AI — GitHub Pages сайты

Бұл папка GitHub Pages-қа жүктеуге дайын статикалық сайт.

## Файлдар
- `index.html` — сайт құрылымы
- `style.css` — дизайн
- `script.js` — батырмалар мен демо форма логикасы

## GitHub Pages-қа жариялау
1. GitHub-та жаңа repository ашыңыз: `Site` немесе `student-ai-site`.
2. Осы папкадағы `index.html`, `style.css`, `script.js` файлдарын repository ішіне жүктеңіз.
3. Repository → Settings → Pages бөліміне кіріңіз.
4. Source: `Deploy from a branch` таңдаңыз.
5. Branch: `main`, folder: `/root` таңдаңыз.
6. Save басыңыз.
7. Бірнеше минуттан кейін сайт мына форматта ашылады:
   `https://username.github.io/repository-name/`

## Домен қосу
GitHub Pages → Custom domain өрісіне өз доменіңізді жазыңыз.
Домен DNS баптауында GitHub Pages берген мәндерді енгізу керек.
